Auxx.ai lets you customize how the interface looks — both at the personal level (theme and font) and at the organization level (brand colors and logo).

## Personal preferences

Each user can set their own theme and font from their account settings.

### Theme

Open the user menu (click your name at the bottom-left of the sidebar) and use the **Theme** toggle to switch between:

| Option | Description |
|--------|-------------|
| **Light** | White background with dark text |
| **Dark** | Dark background with light text |
| **System** | Follows your operating system's preference |

### Font

Go to **Settings > General** and scroll to the appearance section. Choose from:

- **Inter** — Clean sans-serif (default)
- **Manrope** — Friendly sans-serif
- **System** — Your operating system's default font

Click **Update preferences** to save.

## Organization branding

Owners and admins can customize the workspace's visual identity for all members. These settings ensure a consistent look across the team.

### Brand colors

| Setting | Description | Default |
|---------|-------------|---------|
| **Primary color** | Main accent color for buttons, links, and active states | `#4f46e5` (indigo) |
| **Secondary color** | Secondary accent for highlights and badges | `#0ea5e9` (cyan) |

### Logo

Upload your organization's logo. It appears in the workspace header and branding areas.

### Font

Set a default font for the entire workspace. Same options as personal preferences: Inter, Manrope, or System.

### Override controls

For each organization-level setting (colors, font, logo), admins can toggle **Allow User Override**:

- **On** — Members can set their own preference, overriding the org default
- **Off** — The org setting is enforced for all members

This is useful when you want consistent branding across the team but still allow personal preferences where it doesn't matter.
