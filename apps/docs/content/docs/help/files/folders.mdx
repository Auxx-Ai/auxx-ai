title: Folders

---

Folders let you organize files into a hierarchy. You can nest folders, move files between them, and navigate using breadcrumbs.

## Create a folder

1. Go to **Resources > Files** in the sidebar
2. Click **New Folder** in the top-right
3. Enter a name for the folder
4. Click **Create Folder**

![Create New Folder dialog with name input](/images/files-new-folder.png)

The folder appears in the file table. To create a subfolder, navigate into the target folder first, then click **New Folder** again.

## Navigate folders

- **Open a folder** — Double-click it in the file table, or right-click and select **Open Folder**
- **Go back** — Use the breadcrumbs at the top of the page to jump to any parent folder
- **Root** — Click **Files** in the breadcrumb to return to the top level

The **Path** column in the file table shows where each item lives in the folder hierarchy.

## Rename a folder

1. Right-click the folder (or click the three-dot menu on the row)
2. Select **Rename**
3. Type the new name
4. Press **Enter** to save

![Folder context menu showing Quick View, Rename, Open Folder, and Delete options](/images/files-folder-menu.png)

## Move files and folders

Drag and drop files or folders in the table to move them. You can drag items onto a folder row to move them inside it.

## Delete a folder

1. Right-click the folder (or click the three-dot menu)
2. Select **Delete**
3. Confirm the deletion

Deleting a folder removes it and all files inside it. A confirmation dialog appears before the action is carried out.
