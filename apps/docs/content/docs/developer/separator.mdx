---
title: <Separator />
description: Visual divider component for Auxx app dialogs.
---

A visual divider for separating content sections.

```tsx
import { Separator } from '@auxx/sdk/client'

<TextBlock align="left">Section one</TextBlock>
<Separator />
<TextBlock align="left">Section two</TextBlock>

<Separator orientation="vertical" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Divider direction |
